# GitHub Copilot

GitHub Copilot is an AI-powered code completion tool that helps developers write code faster and with less effort. Built on OpenAI's models and trained on billions of lines of code, it provides intelligent suggestions directly in your editor.

## Features

- **Inline Completions** — Get real-time code suggestions as you type
- **Copilot Chat** — Ask questions about your codebase in natural language
- **Multi-file Context** — Understands your entire project for better suggestions
- **Code Review** — AI-assisted pull request reviews
