# Frontend Developer Roadmap

Follow this step-by-step path to master modern frontend web development. Start from the web basics and work your way up to professional frameworks and performance tuning.

## Phase 1: The Core Web Basics

Before using any framework, you must have a solid grasp of how pages are styled, structured, and run.

<RoadmapNode title="Internet & Web Basics" time="1 week" resources="MDN Web Docs: How the Web Works, HTTP Crash Course">
  Learn how web browsers load pages, what HTTP/HTTPS protocols are, how DNS works, and what domain names and hosting are.
</RoadmapNode>

<RoadmapNode title="HTML5 Semantic Markup" time="1-2 weeks" resources="W3Schools HTML Tutorial, freeCodeCamp Responsive Web Design">
  Learn how to structure web pages with accessibility and SEO in mind. Master semantic tags like header, nav, main, article, section, footer, and form handling.
</RoadmapNode>

<RoadmapNode title="CSS3 & Layouts" time="2-3 weeks" resources="CSS Tricks Guides, Flexbox Froggy, Grid Garden">
  Master modern responsive styling. Understand the box model, Flexbox layouts, CSS Grid, media queries, variables, custom properties, and fundamental typography.
</RoadmapNode>

## Phase 2: Programming the Web

Bring your pages to life and handle dynamic interactions.

<RoadmapNode title="JavaScript (ES6+)" time="4 weeks" resources="JavaScript.info, Eloquent JavaScript Book">
  Learn fundamental programming constructs. Understand variables, functions, closures, scoping, DOM manipulation, asynchronous programming (promises, async/await), fetching APIs, and ES6+ syntax (destructuring, spread, modules).
</RoadmapNode>

<RoadmapNode title="Git & Version Control" time="3 days" resources="GitHub Learning Lab, Git Flight Rules">
  Learn how to collaborate with other developers. Master git init, clone, add, commit, branch, merge, pull requests, and resolving conflicts.
</RoadmapNode>

## Phase 3: Building at Scale

Frameworks make it easier to write maintainable code and manage complex applications.

<RoadmapNode title="React & Next.js" time="6 weeks" resources="React.dev documentation, Next.js Learn tutorial">
  Learn components, props, state, hooks, page routing, server/client component division, and server-side rendering (SSR/SSG).
</RoadmapNode>

<RoadmapNode title="Tailwind CSS & Styling Systems" time="1 week" resources="Tailwind CSS documentation">
  Adopt utility-first CSS styling to speed up UI prototyping. Understand how configuration, responsiveness, and dark modes work with Tailwind.
</RoadmapNode>
