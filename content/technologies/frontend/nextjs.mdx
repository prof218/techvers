# Next.js

Next.js is the leading full-stack React framework, created by Vercel. It provides the best developer experience for building modern web applications with features like server-side rendering, static generation, and API routes out of the box.

## Key Features

### App Router (v13+)
The App Router introduces React Server Components, nested layouts, streaming, and a new file-based routing system.

### Server Components
Render components on the server by default, reducing JavaScript bundle size and improving performance.

### Data Fetching
Fetch data directly in Server Components with native `fetch` and automatic caching.

## Architecture

Next.js supports multiple rendering strategies:
- **SSR** — Server-Side Rendering for dynamic content
- **SSG** — Static Site Generation for maximum performance
- **ISR** — Incremental Static Regeneration for stale-while-revalidate
- **Edge** — Edge Runtime for globally distributed compute
